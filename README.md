# Available Scripts

- Clone the repo

  ```bash
  $ git clone https://github.com/francislagares/react-ui-system.git
  ```

- Install the dependencies by running the following command.

  ```bash
  pnpm install
  ```


- To launch Storyobook:

  ```bash
  pnpm storybook
  ```
